Frequently asked
questions

How does Polaris ensure the safety of it's users?

The Polaris bot can only send messages in discord public channels. If a user ever receives a Direct Message (dm) from Polaris bot, it is spam and should not be trusted. To ensure information secrecy, only the user who calls the sensitive command is able to see Polaris' answer.

Can I link more than one wallet per discord account?

In order to ensure data safety and lower the possibility of system exploitation, a Polaris user can only store 1 wallet address per blockchain.

Can I subscribe to an users plan and to a projects plan at the same time?

Yes and the subscriptions have no influence over each other.
